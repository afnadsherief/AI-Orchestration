# ADR-0001 — Repository Contract

## Status

Accepted

## Decision

Separate organizational governance from implementation.

## Responsibilities

### AI-Orchestration

Owns:

- Architecture
- Governance
- Shared Infrastructure
- Agent Contracts
- Product Registry

### AI-Playbook

Owns:

- Standards
- RFCs
- ADRs
- Prompt Library
- Documentation
- Knowledge Management

## Consequences

Projects remain independent while sharing common standards and infrastructure.
